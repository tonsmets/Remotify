Remotify

To use this Spotify Remote, install the app! Or write your own application. The server accepts HTTP Post requests. Simply send a post request to this url and port. Use the parameter-key "action" with one of the following parameter-values. It's just the number!

  • 0 Pause
  • 1 Play
  • 2 Previous track
  • 3 Next track
  • 4 Get current track
